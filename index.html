COMPUTER COMPONENTS

BIOS

CPU

Central Processing Unit
The CPU is the brain of the computer that executes instructions and performs calculations. It processes data and controls the operations of all other components in the system.

ROM

Remove Read-only memory
CMOS is a battery-powered memory chip that stores system configuration data and keeps track of time and date even when the computer is powered off.

RAM

Random Acess Memory
RAM provides temporary, high-speed storage for data and programs currently in use. It allows the CPU to quickly access frequently needed information.

CMOS

Complementary Metal-Oxide Semiconductor
CMOS is a battery-powered memory chip that stores system configuration data and keeps track of time and date even when the computer is powered off.

STORAGE

Storage
Storage devices like hard drives and HDDs provide long-term data storage. They retain information even when the computer is powered off, storing the operating system, programs, and user files.

POWER

Power Unit
The PSU converts AC power from the wall outlet into DC power that computer components can use. It provides stable power distribution throughout the system.
